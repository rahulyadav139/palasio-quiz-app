## Tech Stack

**Client:** NextJS

**Server:** MongoDB

**Authentication:** Iron-session, Password Encryption with Bcrypt


## Functionalities

- User Account
- Quiz Categories
- Dashboard with stats
- Attempted Quiz History
- League Board
- Loader, Toast
